FanTune — Spectral Fan Modulator  v1.0.0
=========================================

WHAT IS FANTUNE?
FanTune recreates the authentic "singing into a box fan" vocal effect — the
choppy, robotic, amplitude-modulated sound, NOT modern Auto-Tune. Three
blades chop the vocal signal at the blade-pass frequency, creating the
iconic "bits of voice removed" texture.

GEOMETRIES
  GRILLE  — Lips on the front guard. Hard chop, deep AM, strongest blade-pass.
  BEHIND  — 6-12" from intake side. Softer chop, more motor hum, the
            nostalgic rear-fan sound.

FORMATS INSTALLED
  VST3   — 64-bit VST 3 audio plugin
  CLAP   — CLAP audio plugin (CLever Audio Plugin)
  Standalone — Direct audio playback + MIDI input (Windows/macOS only)

SYSTEM REQUIREMENTS
  Windows: 10 or later (64-bit)
  macOS:   11 Big Sur or later (Intel x64 + Apple Silicon arm64 universal)
  Linux:   glibc 2.31+, x86_64 (DEB/RPM)
  RAM:     256 MB
  Disk:    50 MB

INSTALLATION LOCATIONS
  Windows: C:\Program Files\Common Files\VST3\FanTune.vst3
           C:\Program Files\Common Files\CLAP\FanTune.clap
           C:\Program Files\FanTune\FanTune.exe (Standalone)
           %USERPROFILE%\Documents\FanTune\Presets\

  macOS:   /Library/Audio/Plug-Ins/VST3/FanTune.vst3
           /Library/Audio/Plug-Ins/Components/FanTune.component
           /Library/Audio/Plug-Ins/CLAP/FanTune.clap
           /Applications/FanTune.app (Standalone)
           ~/Documents/FanTune/Presets/

  Linux:   /usr/lib/vst3/FanTune.vst3/
           /usr/lib/clap/FanTune.clap
           /usr/local/bin/fantune (Standalone)
           ~/.local/share/FanTune/Presets/

FIRST STEPS
  1. Load FanTune on a vocal track in your DAW
  2. Set SPEED to taste (6-24 Hz rotation)
  3. Set DISTANCE to GRILLE or BEHIND
  4. Set WIDTH to control chop depth
  5. Adjust PITCH correction for scale-snapping
  6. Load one of the factory presets to hear the range
